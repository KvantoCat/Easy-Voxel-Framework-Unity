
# Easy-Voxel-Framework-Unity

A framework for rendering large voxel scenes for Unity on the GPU

## Installation

To install the framework, run the designated command in the Assets folder of your Unity project. For creating a test scene, insert a Camera object, attach the two scripts located in the Examples folder to the Camera, and adjust the necessary parameters in the Inspector

```bash
  git clone https://github.com/KvantoCat/Easy-Voxel-Framework-Unity.git
```
